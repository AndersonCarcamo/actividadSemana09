# actividadSemana09

## Pasos para correr:

1. Para crear el ambiente virtual:

```shell
python3 -m venv env
```

2. Entrar al enviroment:

  - En Microsoft: .\env\Scripts\activate
  - En Linux: source env/bin/activate
 
3. Instalar librerias:

```shell
pip install -r requirements

```

4. Ejecutar el programa
